# Week 1 — Resources

Free, public, no signup unless noted.

## Pipeline + funnel reading

- **Patrick McKenzie — "Salary Negotiation: Make More Money, Be More Valued"** (free essay, the canonical reference):
  <https://www.kalzumeus.com/2012/01/23/salary-negotiation/>
- **Haseeb Qureshi — "Ten Rules for Negotiating a Job Offer"** (free):
  <https://haseebq.com/my-ten-rules-for-negotiating-a-job-offer/>
- **levels.fyi** — public compensation data, filtered by company / level / city:
  <https://www.levels.fyi/>
- **Tech Interview Handbook** (open-source, GPL-licensed sibling):
  <https://www.techinterviewhandbook.org/>

## Recruiter and ATS context

- **Greenhouse "Inside Greenhouse" blog** — free company-side perspective on how recruiters actually work:
  <https://www.greenhouse.com/blog>
- **The "Why Resumes Get Filtered Out" archive on AskAManager** (free, the canonical "what HR thinks" resource):
  <https://www.askamanager.org/category/resumes>
- **Jobscan ATS basics** (free article tier — they're a paid product but the basics are public):
  <https://www.jobscan.co/learn/applicant-tracking-systems-101>

## Target-list research

- **Wellfound (was AngelList Talent)** — startup job board, free to browse: <https://wellfound.com/jobs>
- **Hacker News "Who's Hiring?"** — first weekday of every month, free, the highest-signal job board on the web: <https://news.ycombinator.com/submitted?id=whoishiring>
- **Glassdoor reviews** — read with skepticism; trends matter more than individual reviews: <https://www.glassdoor.com/>
- **Blind** (free read-only) — anonymous engineer-to-engineer conversation about specific companies: <https://www.teamblind.com/>

## Compensation data sources

- **levels.fyi** (above) — best for tech-company comp.
- **Bureau of Labor Statistics OOH** (US, free, government source for non-tech roles): <https://www.bls.gov/ooh/>
- **PayScale** (free tier with signup): <https://www.payscale.com/>
- **H1B Salary Database** (US-specific, free, public records): <https://h1bdata.info/>

## Tools you'll set up this week

- A **spreadsheet tool** — Google Sheets (free) is fine, Notion's database view works too, Airtable's free tier is fine.
- A **calendar** for tracking interview schedules — Google Calendar is fine.
- A **notes tool** for post-interview debriefs — Apple Notes / Obsidian / Notion / a Markdown file in your portfolio repo all work. Pick one and stick with it.
- A **email signature** that includes your GitHub URL.

## Glossary

| Term | One-line definition |
|------|---------------------|
| **Funnel** | The pipeline from outreach → offer; each stage has a conversion rate < 1.0 |
| **Recruiter screen** | The first 30-min call with a recruiter (not hiring manager). Logistical + culture-fit basics. |
| **Hiring manager screen** | A 30-60 min call with the engineer who'd manage you. Often a soft technical check. |
| **Tech screen** | A 45-60 min coding interview, usually remote, often on shared editor. |
| **Onsite** (or "virtual onsite") | The full loop. 4-6 hours. Multiple interviewers. Today usually remote. |
| **Coding** | Algorithm-style problem solving. C2 prepares you for this. |
| **System design** | Architecture conversation at senior+ levels. Junior interviews touch it lightly. |
| **Behavioral** | "Tell me about a time…" — STAR-format answers. |
| **Hiring committee** | A panel that reviews all interviewer feedback and decides. Larger companies use these. |
| **Leveling** | Where you land (L3, L4, IC5, etc.) — affects scope and comp by a wide margin. |
| **TC** | Total compensation: base + bonus + equity grant divided by vest period. |
| **RSU** | Restricted Stock Units — equity that vests over time (usually 4 years, often a 1-yr cliff). |
| **Sign-on bonus** | One-time cash, usually paid at offer; "clawback" if you leave within 1 year. |
| **ATS** | Applicant Tracking System — Greenhouse, Lever, Workday, Ashby. Your resume passes through one. |
| **Pipeline** | The simultaneous in-progress conversations with multiple companies. |
| **Funnel math** | Predicting expected offers from a given amount of outreach. |
| **Lateral move** | Switching companies at the same level — different from promotion. |
