# Week 8 — Challenges

1. **[The Senior-Drift Drill](challenge-01-senior-drift.md)** — rehearse the recovery script for when an interviewer pushes the design above your level, so you surface the gap audibly instead of bluffing (~60 min)
2. **[Build the Rubric, Score Yourself](challenge-02-the-rubric.md)** — write the new-grad-passing rubric in the format real interviewers use, then score one of your exercise write-ups against it (~60 min)

Optional but high-leverage. The first inoculates you against the most common round-derailing moment; the second hands you the scoring target you're otherwise designing against blind. Both pay off in every onsite for the rest of the cycle.
