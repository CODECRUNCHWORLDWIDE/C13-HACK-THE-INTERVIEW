# Week 9 — Challenges

1. **[The Hint-Ego Drill](challenge-01-the-hint-ego-drill.md)** — a full mock with a human partner who inserts hints whether you need them or not, so you rehearse taking each one cleanly, no defending, no ego (~45 min)
2. **[The Over-Engineering Trap](challenge-02-the-over-engineering-trap.md)** — solve the same problem twice, once "production-quality" and once as simply as possible, and let the side-by-side comparison calibrate how clever the round actually wants you to be (~60 min)

Optional but high-value. Each targets a failure mode the solo exercises can't fully reach — the human-delivered hint and the software-engineer's instinct to over-build. Run them after the three exercises have landed.
