# Week 7 — Exercises

Three rehearsal tasks. This week is text-heavy by design — almost no code — so the exercises build the non-substitutable skills: planning the day, deflecting the line-crossing question, and debriefing your own loop before the memory decays. Each produces an artefact you carry into your real onsite.

1. **[Exercise 1 — Mock Day Script](exercise-01-mock-day-script.md)** — turn the generic onsite shape into a specific, dated, hour-by-hour run-of-show for your next loop (~90 min)
2. **[Exercise 2 — Deflection Rehearsal](exercise-02-deflection-rehearsal.md)** — rehearse and record your responses to the seven hostile-question patterns and the eleven EEOC-protected categories, with a peer asking them cold (~90 min)
3. **[Exercise 3 — Debrief Self-Eval](exercise-03-debrief-self-eval.md)** — run the candidate-side debrief within 2 hours of a loop, scoring every round, gap, and read of the room (~60 min)

## How to work them

Do them in order — but mind the timing, because two of them are gated on events.

Exercise 1 is the foundation: the run-of-show feeds the morning of the loop and feeds Exercise 3. Do it first, this week.

Exercise 2 needs a peer who is comfortable asking the illegal questions out loud in roleplay. Confirm that with them before you book the hour — some people find it uncomfortable, and you want a willing partner, not a reluctant one.

Exercise 3 is run *after* the loop — the mini-project's 4-hour mock or a real onsite — and the 2-hour window is non-negotiable. By 12 hours out the granular memory has decayed; by 48 hours the document is half-fiction. Have the file open before the loop ends so you can start the moment you sit down.

If your real onsite is on Friday, prioritise Exercises 1 and 2 — they prepare you for the loop. Exercise 3 happens after, so it never blocks the loop itself.
