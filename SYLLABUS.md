# C13 · Hack the Interview — Syllabus

**Format:** 10 weeks · self-paced · open-source (GPL-3.0). Each week = lectures, exercises, challenges, a mini-project, homework, and a quiz.

**Prerequisites:** C2 · CrunchTime The Code completed or running in parallel — C13 teaches the *process* around the problems, not the problems themselves. A resume that exists (it can be bad; Week 2 fixes it). Comfortable running a job search on your own schedule.

**Assessment (honor-based):** tick each lesson complete in the reader; finish the weekly mini-project and quiz. A certificate is issued on 100% completion to One-Stop members.

| Week | Theme | Mini-project |
|------|-------|--------------|
| 1 | The hiring pipeline — funnel math, target list, outreach | The tracking dashboard |
| 2 | The resume that survives the ATS — parse-clean, keyword-ranked | Resume v2: ATS-tested, peer-reviewed |
| 3 | LinkedIn, GitHub, and the public surface | Polished public surface, peer-reviewed |
| 4 | The recruiter screen survival kit | Recorded mock recruiter screen with self-feedback |
| 5 | The hiring manager screen | Recorded mock hiring-manager screen with self-feedback |
| 6 | The technical phone screen — coding on a shared editor | Recorded mock technical phone screen |
| 7 | The onsite — day-of logistics across four to six hours | A 4-hour mock onsite with peers |
| 8 | System design 101 — FAANG-style rounds for new-grad and L4 | TinyURL clone: end-to-end design write-up |
| 9 | The live coding interview — solving out loud on the clock | Record yourself coding, self-grade against a rubric |
| 10 | Offers, negotiation, and compensation literacy | The C13 capstone (capstone) |

**Outcome:** run a structured hiring pipeline end to end — from target list and clean resume through every interview round to a negotiated offer.
